## Project Description
This project demonstrates unit testing and integration testing of a Python
banking system using PyTest. The system includes deposit, withdraw, interest
calculation, loan eligibility, and fund transfer functionalities.

## How to Run Tests
1. Install dependencies:
   pip install -r requirements.txt

2. Run all tests:
   pytest -v

3. Generate HTML report:
   pytest --html=report.html -v

## GitHub Actions
GitHub Actions automatically runs all unit and integration tests when code is
pushed to GitHub. A green tick means all tests passed.
